WSL: 输入xcalc, 报错
============================================================================
问题
----
```bash
suw@ubuntu2204:~$ xcalc
Warning: Cannot convert string "8x13" to type FontStruct
Warning: Cannot convert string "-adobe-symbol-*-*-*-*-*-120-*-*-*-*-*-*" to type FontStruct
```

-----------------------------------------------------------------------
解决方案
--------
```bash
suw@ubuntu2204:~$ export LANG=C
```
